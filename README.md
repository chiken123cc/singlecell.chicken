Analysis pipelines for the Chicken Single Cell Atlas 

1. Introduction

We present a comprehensive chicken single-cell atlas comprising 1.57 million cells across 36 tissues and 157 cell types. This resource reveals the evolutionary dynamics of avian cell types, particularly immune cells, and provides new insights into immune system evolution across amniotes.
